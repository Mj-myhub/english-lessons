## 📝 ENTRY TEST (working in production!)

40 multiple-choice questions:
- **A1 Beginner:** Q1-10 (basic grammar, verbs, articles)
- **A2 Elementary:** Q11-20 (past tense, modals, comparatives)
- **B1 Intermediate:** Q21-30 (conditionals, present perfect, passive)
- **B2 Upper-Intermediate:** Q31-35 (inversion, phrasals)
- **C1 Advanced:** Q36-40 (collocations, idioms, formal register)

**Scoring (cumulative):**
- 36+ correct → C1 Advanced
- 30-35 → B2 Upper-Intermediate
- 22-29 → B1 Intermediate
- 14-21 → A2 Elementary
- 0-13 → A1 Beginner

**Lead capture flow:**
1. Student takes 40 questions
2. Lead form: name + email + phone (optional) + consent
3. Submission emails Maj via Web3Forms with: name, email, phone, level, score, time taken, level breakdown, all 40 answers
4. Student sees beautiful result screen with CEFR level, description, score stats, CTAs to book trial or see pricing

## 🚨 IMPORTANT TECHNICAL NOTES

1. **All emojis use HTML entities** (`&#127881;` etc.) for cross-platform compatibility
2. **Logo CSS uses `border-radius: 8px`**, NOT 50% (50% caused white frame issue)
3. **Fade-in animations** use IntersectionObserver, NOT `opacity:0` (which broke sections if JS failed)
4. **Language switching:** body class toggle + CSS `[data-it] { display: none !important; }`
5. **All buttons use pill shape** (`border-radius: 50px`)
6. **Web3Forms key is LIVE:** `6843cbdc-c4d3-4d2b-963f-c7811e8a56e1` — every test submission emails Maj
7. **File size:** ~143KB, single self-contained HTML file
8. **JS validates cleanly** with `node --check`
